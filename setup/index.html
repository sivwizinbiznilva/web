So you bought a new computer?

Is it OSX?

  1. Download and install Xcode from App Store
  2. Install Iterm2
    • Modify keybindings for i3-like experience
  3. Install shiftit or amethyst
  4. Remap caps lock to escape using pckeyboardhack
    • change from 51 to 53
  5. Install homebrew
  6. Install nginx
    $ brew install nginx
    Then start nginx
    $ sudo nginx
  7. Get node.js
  8. Clone your dotfiles
    $ git clone http://github.com/sivwizinbiznilva/dotfiles
  9. Move your dotfiles into ~/.dotfiles
  10. Setup symlinks
  11. Git clone vundle
  12. Install LaTeX
  13. Clone vimfiles
  14. Clone Silva (vimrc, vimperatorrc)
  15. Clone build script
  16. cd into build script dir
  17. npm install ejs, highlight.js, path, marked
  18. don't forget:
    alias "buildweb"="cd ~/github/course-build/; node build.js /Users/Silva/github/web /Users/Silva/github/web/template.ejs Hippo!"
  19. If you copy your key, you must change the ownership
    chown *user:group* ~/.ssh/id_rsa*
